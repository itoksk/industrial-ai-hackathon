02

課題発見の手法（QC検定4級・3級レベル）

📖 品質管理の基本手法を学びましょう

工業高校生として知っておくべき、実践的な分析手法です。

なぜなぜ分析（5 Whys）

問題の根本原因を探るため、「なぜ？」を5回繰り返す手法。トヨタ生産方式で有名です。

🎯 実践の流れ
  1. 問題を明確に定義する
  2. 「なぜその問題が起きたか？」を問う
  3. 答えに対してさらに「なぜ？」を問う
  4. 5回程度繰り返し、根本原因に到達
  5. 根本原因に対する対策を立案

💡 実例で理解しよう

問題：不良品が多い
なぜ1→検査が不十分だから
なぜ2→検査基準が曖昧だから
なぜ3→マニュアルが古いから
なぜ4→更新体制がないから
なぜ5→責任者が不明確だから

根本原因：品質管理体制の不備
課題：品質管理責任者の任命とマニュアル更新体制の構築

特性要因図（フィッシュボーン図）

問題の要因を体系的に整理する図。魚の骨のような形で、4M（Man, Machine, Material, Method）の観点から分析します。

🎯 作成手順
  1. 解決したい問題を魚の頭（右端）に書く
  2. 大骨として主要因（4M）を配置
  3. 各主要因に対して小骨（詳細要因）を追加
  4. ブレインストーミングで要因を洗い出し
  5. 重要な要因を特定して対策検討

💡 4Mの観点

  • Man（人）：スキル、経験、教育、モチベーション
  • Machine（設備）：機械、道具、コンピュータ
  • Material（材料）：原材料、部品、情報
  • Method（方法）：手順、プロセス、ルール

パレート図（80-20の法則）

問題の80%は20%の要因から生じるという法則。重要な要因から優先的に対処します。

🎯 作成と活用
  1. データを収集し、要因別に分類
  2. 頻度や影響度の大きい順に並べ替え
  3. 棒グラフと累積曲線を作成
  4. 累積80%までの要因を重点項目に
  5. 重点項目から優先的に改善

ブレインストーミング

チームで自由にアイデアを出し合う創造的問題解決法です。

🎯 4つの原則
  1. 批判禁止：どんなアイデアも否定しない
  2. 量重視：質より量を重視する
  3. 自由奔放：突飛なアイデアも歓迎
  4. 便乗歓迎：他人のアイデアに乗っかる
