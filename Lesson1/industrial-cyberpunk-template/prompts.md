# Industrial Cyberpunk Design Generation Prompts

## 基本構造生成プロンプト

### 初期HTML生成
```
工業的サイバーパンクデザインのWebページを作成してください。

要件：
- 背景: 純粋な黒 (#000)
- メインカラー: 警告イエロー (#FF0)
- アクセント: 危険レッド (#F00)、テクニカルグリーン (#0F0)
- フォント: Space Mono (mono), Bebas Neue (display), Inter (body)
- カーソル: crosshair
- 技術的・工業的な雰囲気

必須要素：
1. 警告ストライプ（黄黒の斜め縞）
2. ターミナル風のヘッダー
3. ASCIIアート装飾
4. タイムラインセクション
5. データグリッド/テーブル
6. ステータスインジケーター

出力：
- 完全に動作するHTML（インラインCSS・JS含む）
- 外部ライブラリ不要
```

### ハッカソンサイト生成
```
90分の工業AIハッカソン用のWebサイトを作成してください。

構造：
1. ヒーローセクション
   - 巨大なイベント名
   - カウントダウンタイマー
   - 警告ストライプ装飾

2. タイムライン
   - 時刻表示（黄色）
   - 緊急フェーズ（赤）
   - 各フェーズの説明

3. ストーリーセクション
   - 工場シナリオ
   - 問題提示
   - ASCIIボーダー

4. 評価基準
   - スコアグリッド
   - 配点表示
   - 重要項目ハイライト

5. サポートセクション
   - リソーステーブル
   - ツールリンク
   - ステータス表示

スタイル：
- 黒背景 + 黄色メイン
- モノスペースフォント多用
- 警告/危険表示
- 技術的な装飾
```

## コンポーネント別プロンプト

### タイムライン生成
```
ハッカソンのタイムラインコンポーネントを作成。

仕様：
- 左側: 時刻（黄色、太字、Space Mono）
- 右側: 内容説明
- 左ボーダー: 4px solid #FF0
- 緊急項目: 赤色 + 点滅アニメーション
- グリッドレイアウト: 100px 1fr

HTML構造：
<div class="timeline">
  <div class="timeline-item">
    <div class="time">09:00</div>
    <div class="content">
      <h3>開会式</h3>
      <p>説明</p>
    </div>
  </div>
  <div class="timeline-item urgent">
    <div class="time">10:30</div>
    <div class="content">
      <h3>締切</h3>
    </div>
  </div>
</div>
```

### データテーブル生成
```
工業的なデータテーブルを作成。

デザイン：
- ヘッダー: 黄黒ストライプ背景
- セル: 1px solid #333
- ホバー: 背景色変更
- フォント: Space Mono
- ステータス表示: 色分け（緑=正常、赤=エラー、黄=警告）

構造：
<table class="data-table">
  <thead>
    <tr>
      <th>項目</th>
      <th>ステータス</th>
      <th>値</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>センサー01</td>
      <td class="status-ok">正常</td>
      <td>98.5%</td>
    </tr>
  </tbody>
</table>
```

### 警告コンポーネント
```
緊急警告表示コンポーネントを作成。

要件：
- 赤いボーダー: 2px solid #F00
- 背景: rgba(255,0,0,0.1)
- アイコン: ⚠ または ⚡
- 点滅アニメーション
- 太字テキスト
- 大文字表記

CSS：
.warning-box {
  border: 2px solid #F00;
  background: rgba(255,0,0,0.1);
  padding: 1rem;
  animation: warning-pulse 1s infinite;
}
```

## インタラクション追加

### ホバーエフェクト
```
工業的なホバーエフェクトを実装。

ボタン：
- 通常: 黄色背景、黒文字
- ホバー: 赤背景、白文字、グロー効果
- transform: skewX(-10deg) → scale(1.05)

リンク：
- 通常: 白文字
- ホバー: 黄色、下線（波線）
- text-shadow: 0 0 10px #FF0

カード：
- box-shadow: なし → 4px 4px 0 #FF0
- transform: translate(-2px, -2px)
```

### アニメーション
```
技術的なアニメーションを追加。

スキャンライン：
@keyframes scan {
  0% { transform: translateY(-100%); }
  100% { transform: translateY(100vh); }
}
duration: 4s
linear, infinite

タイプライター：
@keyframes typing {
  from { width: 0; }
  to { width: 100%; }
}
steps(40, end)
カーソル点滅付き

グリッチ：
赤とシアンのズレ
clip-path でランダム切り抜き
0.5秒の高速アニメーション
```

## レスポンシブ対応

```
モバイル対応（768px以下）：

調整内容：
- フォントサイズ: 0.9倍
- グリッド: 複数列 → 1列
- タイムライン: 時刻を上に配置
- テーブル: 横スクロール追加
- アニメーション: 簡略化または削除
- 余白: 75%に縮小

維持する要素：
- 色彩スキーム
- 基本的な警告表示
- 重要な情報の可読性
```
