🎯 ミニ演習：統合的な問題解決

シナリオ：ある工場で「製品の納期遅延が月5件から月15件に増加」という問題が発生しています。

ステップ 使用する手法 実施内容 期待される成果
1. 問題の深堀り なぜなぜ分析 「なぜ納期が遅れるのか？」を5回繰り返す 根本的な問題の特定
2. 要因の整理 フィッシュボーン 4M（人・機械・材料・方法）で要因分類 体系的な要因マップ
3. 重点化 パレート図 遅延原因別の件数を分析 優先対策項目の決定
4. 対策立案 ブレインストーミング AIを活用して改善案を生成 実行可能な改善計画

🚀 Google Apps Scriptとは？

GoogleのサービスをJavaScriptで自動化・拡張できるクラウドベースのプラットフォームです。

  • 無料で使える（Googleアカウントがあれば）
  • サーバー不要（Googleのインフラで動作）
  • スプレッドシート、ドキュメント、フォームと連携
  • Webアプリとして公開可能
  • 定期実行（トリガー）設定可能

📝 STEP 1: 最初のGASプロジェクト

  1. Google Driveを開く
  2. 新規 → その他 → Google Apps Script
  3. プロジェクト名を「品質管理アプリ」に変更
  4. 以下のコードを入力：
// 基本的なWebアプリケーション
function doGet() {
  const html = HtmlService.createHtmlOutputFromFile('index')
      .setTitle('品質管理ダッシュボード')
      .setXFrameOptionsMode(HtmlService.XFrameOptionsMode.ALLOWALL);
  return html;
}

// データを返すAPI
function getData() {
  return {
    message: "工場の品質データ",
    不良率: 3.5,
    生産数: 1000,
    更新日時: new Date().toLocaleString('ja-JP')
  };
}

🌐 STEP 3: デプロイと公開

  1. デプロイボタンをクリック
    右上の「デプロイ」→「新しいデプロイ」
  2. 種類の選択
    歯車アイコン → 「ウェブアプリ」を選択
  3. 設定
    - 説明: 品質管理ダッシュボード v1
    - 実行ユーザー: 自分
    - アクセス: 全員
  4. デプロイ実行
    「デプロイ」ボタンをクリック → URLをコピー
⚠️ 重要：初回は権限の承認が必要です。「詳細」→「安全でない...」→「許可」の順にクリック。

🔗 STEP 4: スプレッドシートとの連携

実際のデータをスプレッドシートから読み込む方法：

// スプレッドシートからデータを取得
function getSheetData() {
  // スプレッドシートのIDを指定
  const spreadsheetId = 'YOUR_SPREADSHEET_ID';
  const sheet = SpreadsheetApp.openById(spreadsheetId).getActiveSheet();
  
  // データ範囲を取得
  const dataRange = sheet.getRange(2, 1, sheet.getLastRow() - 1, 4);
  const data = dataRange.getValues();
  
  // パレート分析用にデータを整形
  const paretoData = data.map(row => ({
    要因: row[0],
    件数: row[1],
    累積: row[2],
    累積比率: row[3]
  }));
  
  return paretoData;
}
